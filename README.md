# Loco Landing Page

[![GitHub stars](https://img.shields.io/github/stars/Mahmoud-Hanyy/Loco_Landing-Page)](https://github.com/Mahmoud-Hanyy/Loco_Landing-Page/stargazers)
[![Live Demo](https://img.shields.io/badge/demo-live-green)](https://mahmoud-hanyy.github.io/Loco_Landing-Page/)

**Loco** is my graduation project — a platform that brings Egyptian local brands together in one app. This repository contains the official landing page where users can discover the app's value, explore features, and find installation instructions.

Instead of endlessly searching for local brands on Instagram, Loco helps users find them easily, all in one dedicated place.

**Live Demo:** https://mahmoud-hanyy.github.io/Loco_Landing-Page/

## Features

- Hero with phone mockup showing actual app UI (brand cards, search bar, category pills)
- Stats bar with quick numbers (brands, plans, support, free start)
- 6 animated feature cards with hover effects
- Pricing plans with Gold highlighted as "الأكثر شيوعاً"
- Full-width download CTA with store buttons
- Team avatar cards with initials and color gradients
- Scroll reveal animations for all sections
- RTL and Arabic-first design

---

## Technologies Used

- HTML5
- CSS3 (Flexbox, Grid, animations)
- JavaScript (scroll reveal)
- Google Fonts (Inter)
